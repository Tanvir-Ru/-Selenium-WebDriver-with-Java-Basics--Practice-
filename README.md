# Selenium WebDriver with Java Basics

A beginner-friendly repository demonstrating the core concepts of web automation using **Selenium WebDriver**, **Java**, and **TestNG**.

## 🚀 Features
- **WebDriverManager**: No need to manually download `.exe` files for Chrome/Firefox.
- **Maven**: Structured dependency management.
- **TestNG**: Used for running tests and making assertions.
- **Implicit Waits**: Handling dynamic elements properly.

## 📋 Prerequisites
- **Java JDK 11** or higher.
- **Maven** installed and configured.
- **IntelliJ IDEA** or Eclipse.
